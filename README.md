# Emoji Interpreter

- This app will give you the meaning of an emoji. Select any emoji displayed and get its meaning
- This app is made using React JS

## How to use

- Click on any emoji or enter a emoji
- Get the meaning of the emoji displayed

## Tech Stack

- HTML
- CSS
- React JS
